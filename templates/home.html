About us


Amezal deviation (performance inhancing calculator) has been made by students of Ganapat University as project for subject Application Development. This deviation calculation tool provides a Task Manager system with which you can create tasks and add your employees as task members. A task member can fill up their work data for the work duration. Our tool will generate graphical data visualization according to data entered by the task member and will generate a report. A Task Manager is allowed to view the work report of the members they added specified to each tasks.

Technology Used in this project

This project has been made in python with the Django webapp framework. For frontend HTML5 and CSS3 has been used with the Bootstrap4 framework. For frontend designing Bootstarp studio has been used and for complete webapp devlopment Pycharm tool has been used. Sqlite3 database has been configured with the webapp for database.
